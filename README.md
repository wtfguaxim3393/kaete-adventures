## 📌 Sobre o Projeto

O **Kaete Adventures** é um sistema web de turismo criado para apresentar destinos turísticos, aventuras e experiências de viagem através de uma interface moderna e interativa.

O projeto foi desenvolvido com foco em aprendizado de desenvolvimento web, utilizando tecnologias básicas de front-end e organização de repositório no GitHub.

---

## 🎯 Objetivo

Criar uma plataforma simples e funcional para:

- Apresentar destinos turísticos
- Buscar aventuras
- Navegar entre páginas
- Trabalhar com layout moderno
- Praticar HTML, CSS e JavaScript
- Organizar um projeto web no GitHub

## 🚀 Como Executar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/wtfguaxim3393/kaete-adventures
```

### 2. Entrar na pasta

```bash
cd kaete-adventures
```

### 3. Abrir no VS Code

```bash
code .
```

### 4. Executar

Abra o arquivo **index.html** no navegador.
